# 🎯 Career Guide – Your AI Career Companion

**Career Guide** is an AI-powered chatbot that helps users discover suitable career paths based on their profile. It provides:

- 🔍 Career suggestions with salary & demand
- 🧭 Personalized learning roadmap
- 🎥 YouTube tutorials
- 📚 Online course recommendations
- 💼 Job listings
- 📄 Option to download the full plan as PDF

Built using:
- 🧠 Groq API (LLaMA 3)
- 💬 Streamlit
- 📺 YouTube Data API
- 📥 PDFKit
- 🤗 Deployed on Hugging Face Spaces

## 🚀 How It Works
1. Fill in your education, interests, skills, and preferences.
2. Get 3 career path suggestions with salary info.
3. Select one to see:
   - Learning roadmap (Beginner → Advanced)
   - YouTube tutorials
   - Courses (Coursera, Udemy, edX)
   - Job search links
   - Downloadable career PDF

Check out the configuration reference at https://huggingface.co/docs/hub/spaces-config-reference
