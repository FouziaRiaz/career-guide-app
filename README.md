🚀 Career Guide AI

An AI-powered career guidance platform designed to help users explore suitable career options and receive personalized learning paths, course suggestions, and job recommendations.

🌟 Project Overview
Career Guide AI was developed during a Generative AI Hackathon as a collaborative project. The goal was to build an intelligent assistant that supports users in making informed and confident career decisions.

The system works like a virtual career mentor by understanding user input and generating structured, practical guidance tailored to individual goals and interests.

🏆 Achievement
Selected among the Top 10 projects in the hackathon.

💡 Key Features
🔍 AI-based career suggestions with salary trends and industry demand insights
🧭 Step-by-step learning roadmaps from beginner to advanced level
🎥 Recommended YouTube learning resources
📚 Curated online courses from platforms like Coursera, Udemy, and edX
💼 Job search links relevant to chosen career paths
📄 Downloadable personalized career plan in PDF format

🛠️ Tech Stack
🧠 Groq API (LLaMA 3) – AI reasoning and career generation
💬 Streamlit – Web application interface
📺 YouTube Data API – Learning resource recommendations
📥 PDF generation tools (PDFKit / wkhtmltopdf) – Export career plans
🤗 Hugging Face Spaces – Deployment platform

🚀 How It Works

* Users enter their skills, education, and interests
* The system generates multiple career suggestions with insights
* Each career includes:

  * Learning roadmap (step-by-step)
  * Video tutorials
  * Course recommendations
  * Job search links
  * Downloadable career plan
